# MSDS_DataVizG1Pikachu
# Repository for MSDS outputs
Note: Instructions are specified for Anaconda Jupyter Notebook users. For Jupyter Notebook users, use the .py file. You can use either .py or ipynb, depending on your application.
1. Download and unzip the Compiled files folder into the same folder as the Merged_charts
2. Run Anaconda Prompt
3. Type: cd "C:\Users\FolderPath" Ex. cd "C:\Users\December2025\Documents\DataViz"
4. Type: python Merged_charts.py or your other file name
5. Copy the dash link on any browser: Ex. http://127.0.0.1:8085
6. Dashboard should now run locally.
